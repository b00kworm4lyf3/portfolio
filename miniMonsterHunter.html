Mini Monster Hunter

C++ Game Design Character Design
Completed: December 2023

Overview

I made Mini Monster Hunter as a final project for my C++ data structures class at Portland Community College. The game is a simple text-based monster inventory, where the player can add their own monsters and then battle them in the arena. This was my first large, multi-file C++ project, and I learned a lot while building it.
